Fonctionnalités

Routage

Buffalo utilise la boîte à outils Gorilla pour gérer les routes, sessions, cookies, etc. Il existe sans doute des routeurs plus rapides, mais celui-ci est le plus puissant !

Gabarits

Composez vos vues avec Plush, en utilisant une syntaxe proche de celle de Ruby on Rails. Étendez ses possiblités en créant vos propres fonctions.
Si vous ne l'aimez pas, vous pouvez utiliser le standard html/template ou même votre propre moteur !

Boîte à outils Buffalo

Développer un module pour une application Web débute toujours par les mêmes tâches. Buffalo fournit une boîte à outils simple, la commande buffalo, pour générer de nombreuses parties de votre application et exécuter des tâches courantes.

Vous pouvez même lui ajouter des plugins, en utilisant le langage de votre choix !

Tests

Tester peut être une tâche ennuyeuse : Buffalo vous aide à définir des scénarios de test. Les générateurs d'action, de resource, ou de modèles créent des gabarits prêts à remplir pour vos tests.

Vous pouvez exécuter tous vos tests à l'aide d'une simple commande de la boîte à outils.

Rechargement à chaud du code

Codez, sauvegardez, rafraîchissez la page. Utilisez la commande buffalo dev pour reconstruire votre application, du cœur logique à l'interface utilisateur, et constatez les changements en direct !

Pipeline frontend Optionnel

Utilisez la configuration Webpack pré-générée pour compiler vos composants graphiques : optimisez votre application de fond en comble !

Modèles / ORM Optionnel

Une intégration forte avec pop permet de gérer les bases de données simplement, ainsi que les tâches courantes associées.

Bases de données supportées : MySQL/ MariaDB, PostgreSQL, SQLite.

Tâches / Tâches de fond Optionnel

Si vous êtes famillier des tâches Rake en Ruby, vous vous sentirez comme chez vous en utilisant Grift. Populer une base de données, lancer une tâche de nettoyage sont maintenant à portée de main !

Vous pouvez également faire tourner des tâches de fond, avec la même simplicité.
